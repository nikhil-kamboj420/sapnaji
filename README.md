# 🎂 Birthday Experience Website

A premium, mobile-first interactive birthday celebration website built with modern web technologies.

## 🎯 Features

- **Smooth Animations** - Powered by Framer Motion and GSAP
- **Beautiful UI** - Tailwind CSS with custom color palette
- **Interactive Sections** - Photo gallery, video player, wishes section
- **Responsive Design** - Optimized for mobile and desktop
- **Smooth Scrolling** - Lenis smooth scroll experience
- **Confetti Effects** - Canvas-based confetti animations

## 🛠️ Tech Stack

- **React 18** - UI library
- **Vite** - Build tool & dev server
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **GSAP** - Advanced animations
- **Lenis** - Smooth scrolling
- **React Icons** - Icon library

## 📦 Installation

```bash
npm install
```

## 🚀 Development

```bash
npm run dev
```

## 📦 Build

npm run build
